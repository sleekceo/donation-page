HOW WILL THE FUNDS BE USED?

All funds you contribute would be used to source and collate for political data, design the data for easy consumption, build new features on the website.

  • Create Candidate Timeline - 4th Republic Presidents
  • Create Candidate Timeline - 4th Republic Governors (36 states)
  • Create Candidate Timeline - 4th Republic Senators (36 states)
  • Create Candidate Timeline - 4th Republic House of Representatives Members (36 states)
  • Create Candidate Timeline - 4th Republic House of Assembly Members (36 states)
  • Create Candidate Timeline - 4th Republic Local Government Chairman (36 states)
  • Create Candidate Timeline - 4th Republic Political Aspirants (36 states)
  • State Comparison Page - A page to compare states with each other based on sector. E.g Lagos vs Cross River (Tourism). Oyo vs Kano (Security)
  • INEC Polling Unit Map - Graphical representation of all Polling Units across Nigeria.
  • “How To” videos - Animated videos on how to become an elected official e.g. How to become a President; How to become a governor etc.

Every Naira you donate to Dacurate is accounted for and will be used to build a more knowledgeable and politically responsible Nigeria.

Once donations are received, we would share a breakdown of how the funds have been spent
